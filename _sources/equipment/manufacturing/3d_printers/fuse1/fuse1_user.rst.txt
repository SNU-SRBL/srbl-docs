User Guide
==========

This guide covers everything required for standard users to prepare, start, and post-process prints using the Formlabs Fuse 1 and the Fuse Sift station.

.. note::
   🚧 **Under Construction** 🚧
   
   The step-by-step operating guide is currently being written. It will soon include:
   
   * Pre-flight checklist (optical window check, powder leveling)
   * Uploading jobs via PreForm software
   * Post-processing parts in the Fuse Sift (powder recycling and part extraction)

.. warning::
   You must receive hands-on training from the equipment manager before operating this machine for the first time.
